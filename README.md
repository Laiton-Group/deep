# ⬡ DEEP

3D geometric exploration — WASD movement, free camera, infinite void.

Explore a world of geometric architecture: spires, crystal gardens, colossal arches, neon grids, floating islands, and an abyss gate. Discover hidden secrets scattered across the landscape.

## Controls

| Key | Action |
|-----|--------|
| `W A S D` / `Z Q S D` | Move |
| `Mouse` | Look around (click to lock pointer) |
| `Shift` | Sprint |
| `E` | Interact / reveal secrets |
| `Esc` | Release pointer |
